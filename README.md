# SpaceX Launch Infrastructure & Trend Analysis

## Project Overview
This project performs an end-to-end analysis of SpaceX launch data fetched directly from the REST API. The goal was to identify success trends over time and correlate launchpad infrastructure with mission outcomes.

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas (Data Wrangling), Matplotlib/Seaborn (Visualization), Requests (API Ingestion)
* **Data Source:** SpaceX REST API

## Key Insights
* **Launch Success Rate:** Analysis of mission outcomes from 2006 to present.
* **Infrastructure Efficiency:** Comparison of success rates across different launchpad locations.
* **Recovery Trends:** Visualizing the evolution of fairing and booster recovery attempts.

## How to Run
1. Clone the repo: `git clone https://github.com/Slylil17/SpaceX-Launch-Analysis.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the notebook: `SpaceX_Analysis.ipynb`
